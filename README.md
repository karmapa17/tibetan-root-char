# tibetan-root-char

> Fetch root character from a Tibetan string.

## Install

```
$ npm install tibetan-root-char
```


## Usage

```js
const {getRootChar} = require('tibetan-root-char');
// Good afternoon ! 
getRootChar('ཕྱི་དྲོ་བདེ་ལེགས།');    // ཕ
```
